addEventListenerのリスナー削除

addEventListenerの第3引数のoptionsにはsignalを受け取ることができる。 これを使うことで自動的にリスナーを削除することができる。 


const abort = new AbortController();

clickButtonElem.addEventListener(
  "click",
  () => {
    // なにか
  },
  {
    signal: abort.signal, // ←signalをセット
  }
);

// こうするとリスナーを削除できる。
abort.abort();
