Temporalの確認(Plain系)

PlainDate, PlainTime, PlainDateTimeなどPlain系はタイムゾーンを持たない

PlainDate

Temporal.Now.plainDateISO()で現在日付のインスタンスを取得できます。 


const plainDate = Temporal.Now.plainDateISO();
各種プロパティ
変数 値
year
month
day
inLeapYear
dayOfWeek
Temporal.PlainDate.from で文字列やオブジェクトから日付を生成できます。 

const plainDate2Format = Temporal.PlainDate.from("2026-07-22");
plainDate2Format.toString(); // "2026-07-22"
const plainDate2Object = Temporal.PlainDate.from({ year: 2026, month: 7, day: 22 });
plainDate2Object.toString(); // "2026-07-22"

PlainTime

Temporal.Now.plainTimeISO()で現在時刻のインスタンスを取得できます。 


const plainTime = Temporal.Now.plainTimeISO();
各種プロパティ
変数 値
hour
minute
second
millisecond
microsecond
Temporal.PlainTime.from で文字列やオブジェクトから時刻を生成できます。 

const plainTime2Format = Temporal.PlainTime.from("12:34:56.789123");
plainTime2Format.toString(); // "12:34:56.789123"
const plainTime2Object = Temporal.PlainTime.from({ hour: 12, minute: 34, second: 56, millisecond: 789, microsecond: 123 });
plainTime2Object.toString(); // "12:34:56.789123"

PlainDateTime

Temporal.Now.plainDateTimeISO()で現在日時のインスタンスを取得できます。 


const plainDateTime = Temporal.Now.plainDateTimeISO();
各種プロパティ
変数 値
year
month
day
inLeapYear
dayOfWeek
hour
minute
second
millisecond
microsecond
Temporal.PlainDateTime.from で文字列やオブジェクトからインスタンスを生成できます。 

// Tは省略可
// 日付と時刻を区切るもので、 T、t、空白のいずれかにすることができます。HH が存在するときのみ存在します。
const plainDateTime2Format = Temporal.PlainDateTime.from("2026-07-22T12:34:56.789123");
plainDateTime2Format.toString(); // "2026-07-22T12:34:56.789123"
const plainDateTime2Object = Temporal.PlainDateTime.from({ year: 2026, month: 7, day: 22, hour: 12, minute: 34, second: 56, millisecond: 789, microsecond: 123 });
plainDateTime2Object.toString(); // "2026-07-22T12:34:56.789123"

PlainYearMonth

Temporal.Nowからは生成できないので Temporal.Now.plainDateISO().toPlainYearMonth(); のようにPlainDateを経由して現在年月のインスタンスを取得する。 


  const plainYearMonth = Temporal.Now.plainDateISO().toPlainYearMonth();
各種プロパティ
変数 値
year
month
Temporal.PlainYearMonth.from で文字列からインスタンスを生成できます。 

const plainYearMonth2 = Temporal.PlainYearMonth.from("2026-07");
plainYearMonth2.toString(); // "2026-07"
これ以外にも演算用のメソッドがいろいろ用意されている。
