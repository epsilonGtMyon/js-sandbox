Duration

Durationは期間を表す

生成方法

Temporal.Duration.fromで書式の文字列やオブジェクトからインスタンスを生成できます。 


// 1年
const _1yearFormat = Temporal.Duration.from('P1Y');
console.log(_1yearFormat.toString()); // P1Y
const _1yearObject = Temporal.Duration.from({ years: 1 });
console.log(_1yearObject.toString()); // P1Y

// -1年
const _minus1yearFormat = Temporal.Duration.from('-P1Y');
console.log(_minus1yearFormat.toString()); // -P1Y
const _minus1yearObject = Temporal.Duration.from({ years: -1 });
console.log(_minus1yearObject.toString()); // -P1Y

// 3日と5時間
const _3day5hourFormat = Temporal.Duration.from('P3DT5H');//Tもしくはtで日付と時刻を区切る
console.log(_3day5hourFormat.toString()); // P3DT5H
const _3day5hourObject = Temporal.Duration.from({ days: 3, hours: 5 });
console.log(_3day5hourObject.toString()); // P3DT5H

日付演算の結果として

日付演算の結果としてDurationを取得することができます。 例えば、新年から現在までの期間を取得する場合は以下のように書きます。


            const now = Temporal.Now.plainDateTimeISO();
            const newYear = now.with({month: 1, day: 1, hour: 0, minute: 0, second: 0, millisecond: 0, microsecond: 0, nanosecond: 0});
            const duration1 = now.since(newYear);
            const duration2 = now.since(newYear, { largestUnit: 'months' });

上記のプロパティ

上記のduration1とduration2のプロパティを表示します。 同じ期間であっても、largestUnitの指定によってプロパティの値が変わることがわかります。

名前 duration1の値 duration2の値

months

利用

日付の加算などに利用

Durationは日付の加算や減算に利用できます。


const duration = Temporal.Duration.from({ days: 3, hours: 5 });
const newYear = Temporal.PlainDateTime.from('2026-01-01');
const added = newYear.add(duration);
added.toLocaleString() // 2026/1/4 5:00:00

totalを使って日数、時間などに換算

Durationのtotalプロパティを使うと、日数や時間などに換算できます。


const duration = Temporal.Duration.from({ days: 3, hours: 5 });
console.log(duration.total({ unit: 'days' })); // 3.2083333333333335
console.log(duration.total({ unit: 'hours' })); // 77
console.log(duration.total({ unit: 'minutes' })); // 4620
