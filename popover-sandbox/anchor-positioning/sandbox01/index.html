いろいろ表示させる。

ボタン側に anchor-name でアンカーの名前をつける 名前は--から始まる。 ポップオーバー側にposition-anchor でアンカー先の名前を指定する。

ポップオーバー側にはブラウザ側がデフォルトでスタイルをあててうまく位置が決まらないので調整が必要 


#myButton1 {
  anchor-name: --myPopover1;
}
#myPopover1 {
  position-anchor: --myPopover1;

  bottom: anchor(top);
  left: anchor(right);
}
